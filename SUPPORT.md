# Support

## Questions

For usage questions, open a GitHub Discussion if discussions are enabled. Otherwise, open an issue and choose the question template.

## Bugs

Open a bug report with:

- Package version.
- iOS version and device or simulator.
- A small reproduction or code snippet.
- Expected behavior.
- Actual behavior.

## Feature Requests

Open a feature request with the problem you are trying to solve and the API shape you would expect to use.
